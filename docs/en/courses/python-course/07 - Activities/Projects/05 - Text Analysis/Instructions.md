## Text Frequency Program Instructions

**Objective**: Develop a program that calculates the frequency of each word across multiple text inputs provided by the user. The program should normalize the text by ignoring case and punctuation, thereby providing a clear count of how often each word appears. Additionally, the program should identify words that are common across the different texts.

### Overview

The ability to process and analyze text is a fundamental skill in many areas of software development, including search engines, data analysis, and natural language processing. This program aims to introduce basic text processing concepts such as tokenization, normalization, and frequency analysis.

### Requirements:

#### Core Functionality

- **Text Input**: Prompt the user to enter a series of texts. The number of texts ('n') can be predetermined or specified by the user.
- **Normalization**: Convert all text to lowercase to ensure case-insensitivity. Remove punctuation from the text to focus solely on the words.
- **Word Frequency Calculation**:
  - Split the text into individual words.
  - Count how often each word appears in the text.
  - Use a dictionary to map words to their frequency counts.
- **Identifying Common Words**: Determine which words are common across all provided texts.
- **Display Results**: 
  - For each text, display the word frequencies in a clear and organized manner.
  - Highlight the words that appear in all texts, including their overall frequency across all inputs.

#### Additional Guidelines

- **Comments and Documentation**: Include comments throughout your code explaining your logic and decisions. This is especially important for parts of the code that handle text normalization and frequency calculation.
- **Modular Design**: Structure your program with functions to improve readability and maintainability. For example, separate functions for normalization, splitting text into words, and counting word frequencies can make your code more organized.

### Deliverables

- Submit the complete source code.
- Ensure your code is well-commented and adheres to the specified features and implementation details.
