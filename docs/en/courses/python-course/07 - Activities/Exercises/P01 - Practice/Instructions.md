## Practice Exercises

**Objective**: The goals of this activity is to empower you to: 
- Understand and apply Python programming concepts in solving real-world problems.
- Develop proficiency in data visualization using Matplotlib.
- Enhance your problem-solving skills through a variety of Python exercises.

### Happy coding! I look forward to seeing your solutions and approaches to these exercises!

## Programs

1. **Matplotlib Graph 1**: Plot Linear Equations Graph: Use Matplotlib to plot the graph of a linear equation provided by the user in the form of y=mx+c.
2. **Matplotlib Graph 2**: Plotting Exponential Growth: Use Matplotlib to plot the exponential growth function y = e^x over a range of x values.
3. **Matplotlib Graph 3**: Plotting Sine and Cosine Waves: Use Matplotlib to plot the sine and cosine functions over a range of values from -π to π.
4. **Histogram of Letter Frequency in a String**: Use Matplotlib to plot a histogram showing the frequency of each letter in a user-inputted string. All the letters in the string should be treated in as lower-case.
5. **Collatz Conjecture**: Implement the Collatz conjecture algorithm, which takes a number n and repeatedly applies specific operations until reaching 1, plotting each step's value using Matplotlib.
6. **GCD (Greatest Common Divisor) Calculator**: Implement a program that calculates the GCD of two numbers using the Euclidean algorithm.
7. **Savings Goal Tracker**: Write a script that calculates how many months it will take to save up to a goal, based on monthly savings and a starting amount.
8. **Quadratic Equation Solver**: Write a script that solves quadratic equations (ax^2 + bx + c = 0), handling both real and complex roots.
9. **Pythagorean Triple Checker**: Create a program that checks if three numbers form a Pythagorean triple (a^2 + b^2 = c^2).
10. **Lunar Calendar Converter**: Implement a simple script that converts a given date from the Gregorian calendar to the corresponding date in a lunar calendar.
11. **Angle Converter**: Write a program that converts angles from degrees to radians and vice versa, using the math module for conversion constants.
12. **Multiplication Table Printer**: Write a script that prints the multiplication table for numbers up to N, where N is input by the user.
13. **BMI Calculator**: Write a program that calculates the Body Mass Index (BMI) from user inputted weight and height.
14. **Area of a Polygon Calculator**: Implement a calculator that computes the area of regular polygons (triangles, squares, pentagons) using the math module for trigonometric functions.


#### Additional Guidelines

- **Comments and Documentation**: Include comments throughout your code explaining your logic and decisions. This is especially important for parts of the code that handle text normalization and frequency calculation.
- **Modular Design**: Structure your program with functions to improve readability and maintainability. For example, separate functions for normalization, splitting text into words, and counting word frequencies can make your code more organized.
