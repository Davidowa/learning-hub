Primer contacto con Python. Fundamento de todo lo demás.

## Lo que aprenderás

- Ejecutar Python — intérprete, scripts, notebooks.
- **Variables** (cajas con nombre para guardar valores) y tipos — `int`, `float`, `str`, `bool`.
- Expresiones y precedencia de operadores (el orden en que Python evalúa `2 + 3 * 4`).
- Entrada y salida — `input()`, `print()`.
- **Control de flujo** (elegir qué código se ejecuta): `if` / `elif` / `else`, `while`, `for`.
- Funciones — bloques reutilizables de instrucciones con entradas y salidas.

## Submódulos

- **1er Módulo** — hola mundo, aritmética, primeros scripts (`Code001`–`Code005`).
- **2do Módulo** — variables, cadenas, entrada del usuario (`Code006`–`Code009`, `Example001`, `Exercise001`).
- **3er Módulo** — expresiones, precedencia, comparación (`Code010`–`Code011`, `Exercise002`, diagramas de precedencia).
- **4to Módulo** — condicionales (`Code012`–`Code015`).
- **5to Módulo** — ciclos.

## Analogía sencilla

Un programa en Python es como una **receta**: cada línea es un paso, las variables son los tazones etiquetados sobre la mesa, y el control de flujo es decidir "si la masa está pegajosa, agrega harina" a mitad de la receta.
