# Python

<img src="https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/brands/python.svg" width="48" height="48" alt="Python">

Curso completo de Python — desde el primer `print` hasta apps de escritorio con base de datos. Cada módulo trae código ejecutable, ejercicios cortos y actividades abiertas.

## Para quién es

- Estudiantes nuevos en programación — **no se asume experiencia previa**.
- Ingenieros que vienen de otro lenguaje — pueden ojear módulos 1–2 y enfocarse en 3+.
- Autodidactas — las actividades te dan algo tangible que construir.

Sin soberbia: los tecnicismos aparecen con definición en lenguaje llano y analogías cotidianas.

## Módulos

- **01 · [Bases](01-basics.md)** — sintaxis, variables, control de flujo, funciones.
- **02 · [POO](02-oop.md)** — clases, herencia, encapsulación.
- **03 · [Archivos y rutas](03-paths-files.md)** — lectura/escritura, serialización, CSV/JSON.
- **04 · [SQLite](04-sqlite.md)** — bases de datos relacionales desde Python.
- **05 · [GUI](05-gui.md)** — interfaces de escritorio.
- **06 · [Avanzado](06-advanced.md)** — decoradores, generadores, async, tipado.
- **07 · [Actividades](07-activities.md)** — proyectos que integran todo lo anterior.

## Cómo ejecutar el código

Clona el repositorio y abre la carpeta de cualquier módulo:

```bash
git clone https://github.com/davidowa/learning-hub
cd learning-hub/courses/python-course
python "01 - Basics/1st Module/Code001.py"
```

Se recomienda Python 3.11+.

## Código fuente

[`courses/python-course/`](https://github.com/davidowa/learning-hub/tree/main/courses/python-course) — organizado por módulo, una carpeta por tema, archivos numerados `CodeNNN.py` para ejemplos y `ExampleNNN.py` / `ExerciseNNN.py` para práctica.
