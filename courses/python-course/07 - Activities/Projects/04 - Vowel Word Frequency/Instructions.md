## Vowel and Word Count Frequency Program Instructions

**Objective**: Create a program that counts the number of vowels in a given text input by the user. The program should be case-insensitive and report not only the total count of each vowel and the total count of all vowels combined but also the total number of words in the text input.

## Overview

This program extends basic string manipulation tasks to include both character and word counting. It's an introduction to parsing and analyzing text data, focusing on vowels and word identification, providing a comprehensive overview of handling strings in Python.

### Requirements:

#### Core Functionality

- **Text Input**: Prompt the user to enter a block of text, ranging from a sentence to multiple paragraphs.
- **Normalization and Preprocessing**: Convert the text to lowercase to ensure case-insensitivity and facilitate easier processing.
- **Vowel Counting**:
  - Count the occurrences of each vowel (a, e, i, o, u) within the text.
  - Utilize a dictionary to map each vowel to its respective count.
- **Word Counting**:
  - Determine the total number of words in the text. Words can be assumed to be separated by spaces and punctuation.
  - Consider using Python's string methods or regular expressions to accurately identify words.
- **Display Results**: Present the results in a user-friendly manner, showing counts for each vowel, the total vowel count, and the total word count. For instance:
```python
Total words: 45
a: 10
e: 5
i: 6
o: 8
u: 3
Total vowels: 32
```

#### Additional Guidelines

- **Modular Design**: Break down the program into functions, such as for counting vowels and words, to enhance readability and maintainability.
- **Comments and Documentation**: Thoroughly comment your code to describe the functionality of each part, especially where you implement logic for counting vowels and words.
